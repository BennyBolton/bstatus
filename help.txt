Usage: bstatus [options]

Continuously output a status line defined via a configuration file.

Options:
 -c --config=<file>      Specify an alternative configuration file
 -l --log=<streams:file> Log messages for the streams given to file
 -s --suppress=<streams> Suppress the default output for the streams
 -k --kill-delay=<delay> Specify the delay befour killing children
 -v --version            Display the version of bkeyd
 -h --help               Display this help text

Where applicable, all options above override options in the configuration file
see the bstatus(1) for more information
